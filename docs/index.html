‘돈 되는 수학’

  • 수학, AI, 데이터, 현실 문제… 모두를 하나로 잇는 교육
  • 시험을 위한 공부가 아닌, 현실을 위한 공부를 시작합니다.
  • GitSAM 교육이 왜 존재하게 되었을까요?
  • GitSAM 교육은 왜 의미가 있을까요?

왜 지금 GitSAM인가?

세상은 AI, 데이터, 자동화로 빠르게 융합되고 있습니다.

하지만 교육은 여전히 산업화시대의 과목 구분에 갇혀 있습니다.

학생들은 묻습니다.
> “왜 수학을 배워야 하나요?” 하지만 학교와 학원은 대답하지 못합니다.

시험성적만을 위해 그저 추상적인 개념들을 외우고, 정형화된 문제를 푸는 방식만 반복할 뿐입니다.

그런데 현실은 그렇게 단순한 시험을 출제하지 않습니다.
오히려, 우리는 복잡한 세상을 단순하게 정의할 수 있는 사고력을 배워야 합니다.

GitSAM은 바로 그 ’사고의 힘’을 훈련합니다.

왜 GitSAM을 시작했는가

집단내 “평균 이하 아이큐”에서,
“최고 명문대 최상위권”까지.

그 비결은 암기가 아니라 사고력 훈련이었습니다.

저는 명문대 졸업 후, 긴 시간 교육을 떠나 생존을 위해 완전히 다른 삶을 살아야 했습니다. 그리고 그 시간 동안, 교육이야말로 사회를 구하는 가장 본질적인 시스템임을 절감했습니다.

그래서 돌아왔습니다.
GitSAM은 한 개인의 회복이자, 교육의 본질을 되찾기 위한 진지한 여정입니다.

→ 대표의 편지 전체 보기

GitSAM은 이런 분들에게 가장 적합합니다

✔️ 수학문제는 잘 풀지만, 왜 배우는지는 모르겠는 학생
✔️ Python, AI, 데이터 분석이 궁금하지만 어디서부터 시작해야 될지 모르는 사람
✔️ 단순한 성적이 아닌 현실을 이해하고 해석하는 힘을 기르고 싶은 사람
✔️ 초등학생부터 중장년층까지, 새로운 지식을 통해 삶을 재설계하고 싶은 누구나

수학은 현실을 그려내는 언어다

복잡한 세상은 수학적 언어로 구체화할 수 있고, 현실 문제를 융합적으로 해결하기 위해서는 수학적 모델링이 가장 강력한 도구이다.

GitSAM은 ‘수학’, ‘과학’, ‘AI’, ‘프로그래밍’, ’사회학’이 분리된 영역이 아니라
하나의 언어, 하나의 사고도구로 연결되어 있다는 철학을 바탕으로 교육합니다.
단순히 과목을 가르치는 것이 아니라, 현실을 이해하고 문제를 해결하는 방식을 훈련합니다.

GitSAM이 특별한 이유

’겉모습만 그럴듯한 융합 교육’이 아닙니다.

기존 교육 GitSAM 방식
문제풀이 → 채점 실습 → 분석 → 포트폴리오
암기식 평가 개념 적용 + 현실 문제 해결
과목별 단절 수학, 통계, AI, 과학, 사회학 융합
강사 중심 학생 중심 사고 흐름 피드백
학습 = 시험 학습 = 개인별 문제해결력
결과 = 점수 결과 = 개인별 포트폴리오

GitSAM은 단지 지식을 ‘가르치는’ 곳이 아닙니다.
학습자의 잠재력을 ‘이끌어내는(Educate)’ 공간입니다1.

커리큘럼은 이렇게 구성됩니다

✅ Mentor-Led

  • 교수·박사·연구자가 직접 지도
  • 정답이 아니라 질문을 찾는 사고 과정 중심 교육

✅ Practice-Based

  • 직접 Python으로 데이터를 분석하고 모델을 구성
  • 이론이 아닌 행동 중심의 학습

✅ Real-World Context

  • 교과서가 아닌 현실 문제를 데이터로 직접 해석
  • 추상 개념을 구체화하며, 지식의 자기화 유도

구체적인 학습 흐름

단계 주제 주요 내용 연결 분야
1단계 수학으로 세상 바라보기 주가·소득·실험 데이터를 함수·통계로 분석 기하학, 사회통계
2단계 AI에게 계산 시키기 Python 코드, 반복문, 조건문, 알고리즘 대수학, 컴퓨터사고
3단계 수학으로 예측하기 회귀분석, 최적화, 선형대수 기반 예측모형 설계 미적분, 수치해석
4단계 AI와 함께 문제해결 분류·군집화, Scikit-learn 기반 머신러닝 실습 기계학습, 병렬처리
5단계 캡스톤 프로젝트 “AI에게 가격 예측시키기” 등 자율 주제 종합형 융합 수행

사용하는 기술도 다릅니다

  • Google Colab: 고성능 환경 없이도 원격 실습 가능
  • Jupyter Notebook: 수학과 코드를 통합 학습
  • Python + Pandas + Sklearn: 직접 데이터 분석 및 AI 기술 실습
  • Quarto: 자신의 사고를 포트폴리오로 시각화
  • (추후) 블록체인 기록 시스템: 사고과정과 평가의 투명성 확보

GitSAM은 입시에도 실질적 도움을 줍니다

  • 프로젝트 학습은 논문, 경시, 수행 평가로 확장 가능
  • 포트폴리오 결과물은 학종 포트폴리오, 자소서, 면접 대비 자료로 활용
  • 단순 성적이 아닌, 실력과 논리적 사고의 객관적 증거로 남습니다

단점? GitSAM은 숨기지 않습니다

GitSAM의 교육은 기존과 다릅니다.
그래서 입시 점수처럼 빠르게 성과를 보이기 어렵습니다.
하지만 저는 말할 수 있습니다.

사고의 훈련이 충분히 이뤄지면, 성적은 생각보다 빠르게 오릅니다.
그건 저의 실제 경험입니다.
그래서 저는 믿습니다.
그리고 그 믿음을 실현할 수 있는 공간이 GitSAM입니다.
